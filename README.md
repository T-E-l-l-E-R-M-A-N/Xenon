# Xenon

Atmospheric Metro-inspired music player built with AvaloniaUI.

Xenon experiments with the original ideas behind Metro / Windows Phone design:

Instead of copying modern mobile interfaces, Xenon tries to recreate the feeling of interactive media environments inspired by:

* Windows Phone Metro UI;
* Zune HD;
* early Windows 10 Mobile concepts;
* public information displays and digital signage systems.

## Features

* Panorama-based navigation
* Large animated typography
* Dynamic parallax backgrounds
* Metro-inspired playback interface
* Experimental spatial layout system
* Lightweight dark UI
* AvaloniaUI cross-platform frontend
